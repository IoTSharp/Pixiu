> 在 IoTSharp.SaaS 生态中，本仓库被引用为 **`IoTSharp.Edge.Linux`** —— Linux C 边缘基座与 BasicRuntime 的 C 版宿主。

## 0. 范围

本仓库提供：

- 基于 C 的边缘网关运行时（数据采集 / 边缘计算 / 上报 / 本地控制面板）
- 跨发行版 / 跨架构（x86 / ARM64 / MIPS）构建脚本
- BasicRuntime C 版宿主接口（用于承载 IoTEmBASIC 脚本）
- 串口 / RS485 / Modbus / I2C / CAN / 插件机制等南向能力

不承载：多租户、Copilot、计费、License。
